## Tech Stack MERN

**Client:** React, React Hooks, Bootstrap 5

**Server:** Node, Express, MongoDB

**Deploy:** Heroku, Vercel

**Setup:** Eslint, Prittier, Husky, LiveShare VSC

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## API Reference

### URL Backend
```http
  https://backend-equipo7.herokuapp.com/
```
 

#### Get user item

```http
  GET /users/${legajo}
```

#### Get user Rol

```http
  GET /users/${legajo}/rol
```

#### Get user courses

```http
  GET /users/${legajo}/courses
```

#### Get user software tools

```http
  GET /users/${legajo}/software
```

#### Get user signin

```http
  GET /users/signin
```

#### POST user signup

```http
  GET /users/signup
```
